Project Overview
Developed an interactive Excel dashboard to analyze and visualize sales and inventory data for Vrindha Store. The dashboard empowers management to track sales trends, monitor inventory health, and make data-driven decisions.

Key Features

Data Cleaning & Preparation: Consolidated and cleaned transactional sales and stock data for accuracy.

Core Metrics:

Total sales and revenue

Category- and product-wise sales breakdown

Monthly/weekly sales trends

Inventory status and stock alerts

Top-selling and low-performing products

Interactive Visualizations: Utilized Pivot Tables, Dynamic Charts, and Conditional Formatting to bring insights to life.

Custom Dashboards: Slicers and Filters allow real-time exploration by product, category, or time period.

Tools & Techniques

Microsoft Excel (Pivot Tables, Power Query, Charts, Conditional Formatting, Slicers, Data Validation)

Outcome :

Delivered a visually compelling dashboard connecting daily operations with strategic insights.

Enabled fast, actionable decision-making based on up-to-date sales and inventory data.
